---
title: Installation
description: What to do next
---

To install Butterfly, you can use the following command:
```sh
dart pub global activate butterfly_cli
```
<Info>Do not need to run this command in the project. Its a global command.</Info>

After that, you can run the following command to initialize your project:
```sh
butterfly init
```

The executable is named `butterfly`:

```sh
butterfly --help
butterfly init
butterfly commit
```

This command creates the following directories directly at your project root:

- `models`: Your data models.
- `services`: Your service classes.
- `screens`: Your UI screens.
- `widgets`: Your reusable widgets.

Beyond folders, `butterfly init` also runs an interactive prompt asking whether to add **Core**, **Auth**, and **Router** features — and automatically adds the required butterfly packages as git dependencies.
